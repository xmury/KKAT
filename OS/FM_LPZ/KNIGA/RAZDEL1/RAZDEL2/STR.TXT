1. Двойным щелчком по иконке
2. alt + f4
3. В краткой показывают только имена директорий и папок, а в полной показывают расширения, доступ и дату
4. В нижней части окна. Необходима для выполнения dos команд.
5. При перемещении файл меняет свою директорию, а при копировании он остаётся в свое директории а в 
	указанной создаётся его копия
6. Для выполняния завязанных на них команд
7. Выделенный файл - файл над которым можно сразу производить действия, в отличии от обычного файла, который сперва следует выделить,
 	т.е сделать выделенным
8.      - Новый текстовый файл: shift + f4
	- Вывести файлы с сортировкой по имени: ctrl + f3
	- Вывести на экран в краткой форме: ctrl + 1
	- Переход в корень диска: ctrl + \
	- Просмотр текстового файла: ctrl + q
	- f7
	- Переименовать: shift + f7
	- Удалить директорию: f7
	- Копировать файл: f5
	- Перенести: f6
